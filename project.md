# Project: Regrowth Ecological Simulation Game

## Description

Regrowth is a gamified ecological simulation where players restore degraded land using permaculture techniques, water harvesting, and community labor. The project uses Bevy 0.18 engine with a hexagonal grid-based world system, real-world measurement scales (5cm hexagon tiles), and modular simulation systems for soil, water, plants, organisms, and character labor.

> For per-task descriptions, success criteria, prerequisites, and dependency details, see [task_details.md](task_details.md).
